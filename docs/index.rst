Welcome to mobility-datasets documentation!
===========================================

**mobility-datasets** is a Python package for working with autonomous driving datasets.
It provides a unified interface for loading and working with popular datasets including
KITTI, nuScenes, and Waymo Open Dataset.

Features
--------

* **Unified API**: Consistent interface across different datasets
* **Dual Data Access**: Both pandas DataFrames and typed dataclasses
* **Lazy Loading**: Efficient memory usage with lazy data loading
* **Coordinate Systems**: Built-in coordinate transformation utilities
* **CLI Tool**: Command-line interface for dataset management
* **Type Hints**: Full type annotations for better IDE support

Supported Datasets
------------------

* **KITTI**: KITTI Vision Benchmark Suite
* **nuScenes**: nuScenes autonomous driving dataset
* **Waymo**: Waymo Open Dataset
